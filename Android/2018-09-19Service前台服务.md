# Service前台服务

自定义Service，在onCreate方法中启动为前台服务

```java
    @Override
    public void onCreate() {
        super.onCreate();
//        启动前台服务
        Notification.Builder myBuilder = new Notification.Builder(this);
        myBuilder.setContentIntent(PendingIntent.getActivity(this, 0, new Intent(this, MainActivity.class), 0));
        myBuilder.setAutoCancel(false);
        myBuilder.setSmallIcon(R.mipmap.ic_launcher);
        myBuilder.setTicker("Foreground Service Start");
        myBuilder.setContentTitle("Socket服务端");
        myBuilder.setContentText("正在运行...");
        startForeground(1, myBuilder.getNotification());
    }
```
