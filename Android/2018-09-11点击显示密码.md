2.Activity

```java
public class MainActivity extends AppCompatActivity {

    private TextView tv;
    private EditText et_psw;
    boolean passwordVisible=false;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        initView();
        tv.setOnClickListener(new View.OnClickListener() {

            @Override
            public void onClick(View v) {
//                获取光标最后一位
                int editSelection=et_psw.getSelectionEnd();
                if(passwordVisible){
                    tv.setText("隐藏");
//                    显示密码
                    et_psw.setTransformationMethod(PasswordTransformationMethod.getInstance());
                }else{
                    tv.setText("显示");
//                    隐藏密码
                    et_psw.setTransformationMethod(HideReturnsTransformationMethod.getInstance());
                }
                passwordVisible=!passwordVisible;
//                点击后重新将光标设置到末尾 不然会跳到最前边
                et_psw.setSelection(editSelection);
            }
        });
    }

    private void initView() {
        tv = findViewById(R.id.tv_open);
        et_psw = findViewById(R.id.et_psw);
    }
}
```

# 2.去掉标题栏、全屏显示

`去标题栏`

继承的是``Activity`：

在Activity中添加

```
requestWindowFeature(Window.FEATURE_NO_TITLE);
```

继承的是``AppCompatActivity`:

1.Activity中添加

```
getSupportActionBar().hide();
```

2.ManiFest中修改Activity的主题为

```
android:theme=”@style/Theme.AppCompat.Light.NoActionBar”
```

`全屏显示`

```java
getWindow().setFlags(WindowManager.LayoutParams.FLAG_FULLSCREEN,
                WindowManager.LayoutParams.FLAG_FULLSCREEN);
```
